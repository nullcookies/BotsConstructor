Потомок/предок

Инструкция по использованию.
Проект является смесью говнокода и Asp.Net Core 2.2 MVC.
Для использования нужно:
1) Заменить токен телеграм бота на свой (брать у BotFather)
2) Настойка ip адреса. Для того, чтобы телеграмм знал, куда нужно 
посылать post запросы нужно выполнить строку

BotClient.SetWebhookAsync("<url сервера>").Wait();

	1. Если есть белый ip (тот, который видно из 
	интернета), то заменить строку адрес в скобках на свой внешний ip
	
	BotClient.SetWebhookAsync("186.16.16.27", allowedUpdates: allowedUpdates).Wait();

	2. Если есть только серый ip то можно бесплатно настроить ngrok.
	Заходим на https://ngrok.com/ и регистрируемся. 
	Потом скачиваем программу на компьютер. Открываем консоль в папке 
	с скачанным фалом. В ней вводим команду с сайта 
	
	./ngrok authtoken as64asd8fsa614f..d8fsa61
	
	Потом выполняем 
	
	ngrok http 8080
	
	ngrok выдает ip типа такого https://a8981dc5.ngrok.io.
	Копируем адрес с  https в первый параметр
	
	BotClient.SetWebhookAsync("https://a8981dc5.ngrok.io", allowedUpdates: allowedUpdates).Wait();
	
	
Запускаемся. Готово.
